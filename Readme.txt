* ------- *
* Do file *
* ------- *

**bruhn_gallego_tables_restat.do

This Stata do-file generates the summary statistics and regression results reported in the paper and appendix tables.


* --------- *
* Data file *
* --------- *

**bruhn_gallego_tables_data.dta

This file includes all variables used in the paper. All variables are labeled.
Appendix A provides the corresponding variable definitions.
Appendix B lists the data sources.
